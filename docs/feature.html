remoTree Features

  • Shows graphical tree views for both remote and local file systems.
  • Files and directories can be transferred using simple copy and paste commands.
  • Includes a basic editor for text files.
  • Includes a basic command line shell on the remote system.
  • Can easily create public/private key pairs (optionally with a passphrase) to control access without using system passwords.
